1. Array (Opsional)

Menghapus elemen pertama dari array:

let array = [1, 2, 3, "halo", false, true];
array.shift(); // Menghapus elemen pertama
console.log(array);
// Output: [2, 3, "halo", false, true]

2. Operasi Conditional

Memeriksa apakah cheese adalah cheddar:

let cheese = "cheddar";

if (cheese === "cheddar") {
    console.log("Yay! Cheese available for making cheese on toast.");
} else {
    console.log("No cheese on toast for you today.");
}

3. Operasi Looping

Melakukan looping pada properti dalam objek:

const buku = {
    judul: "Harry Potter and The Philosopher's Stone",
    pengarang: "J. K. Rowling",
    tahun: 1997
};

for (let key in buku) {
    console.log(key + ": " + buku[key]);
}

4. Operasi Try - Catch - Finally

Menangani error dengan try-catch-finally dan menampilkan pesan yang sesuai:

const number = 100, text = 'a';

try {
    console.log(number / text); // NaN
    console.log(a); // ReferenceError karena 'a' tidak didefinisikan
} catch (err) {
    console.log("An error caught");
    console.log("Error message: " + err);
} finally {
    console.log("Finally will execute every time");
}

// Output:
// NaN
// An error caught
// Error message: ReferenceError: a is not defined
// Finally will execute every time
